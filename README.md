# Log-Archive-Tool

## Installation
1. Clone the repository or download the `log-archive.sh` script:
   ```sh
   git clone <project-repo-url>
   cd log-archive
   ```

2. Grant execute permissions:
   ```sh
   chmod +x server-stats.sh
   ```

## Usage
Run the script using:
```sh
./log-archive.sh
```
